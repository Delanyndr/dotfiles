On a fresh intall for an easy and nice setup can use LARBS
curl -LO larbs.xyz/larbs.sh; then run as root: bash larbs.sh

===== Custom from fresh install ===== 
1. Install the following packages:
    a. Xorg: xorg-server xorg-xinit xorg-apps
    b. Driver: xf86-video-<graphics> (to check: lspci | grep -e VGA -e 3d)
    c. Fonts: ttf-dejavu ttf-font-awesome ttf-hack
    d. i3: i3-gaps i3blocks i3lock dmenu termite compton
    e. Utils: xclip libnotify dunst xwallpaper alsa-utils

2. Configure Xorg, i3, etc:
    a. cp /etc/X11/xinit/xinitrc ~/.xinitrc; replace last block with `exec i3`
    b. cp /etc/xdg/termite/config ~/.config/termite/

3. Configure Sound: 
    a. run `alsamixer`
    b. Unmute 'Master' and 'PCM' increase until 0dB gain

3. Notes
    xf86-input-synaptics | driver for touchpad
    `startx` then add 'status_command i3blocks' to ~/.config/i3/config 
    HiDpi scaling: 
        'Xft.dpi: 192' to ~/.Xresources or xrandr --dpi 192
        add GDK_SCALE=2 to .profile
    xrandr --output <screen> --mode <resolution> (add to .xinitrc)
    xwallpaper --maximize /path/to/wallpaper
    lxapperance for gtk themes
