Main Package List (run bootstrap script after this):
    Xorg:   xorg-server xorg-xinit xorg-apps
    Audio:  pulseaudio pulseaudio-alsa pulsemixer 
    Driver: xf86-video-<graphics> (to check: lspci | grep -e VGA -e 3d)
    Fonts:  ttf-dejavu otf-font-awesome ttf-hack
    i3:     i3-gaps i3blocks i3lock dmenu termite compton
    System: sysstat htop acpi lxappearance neofetch 
    Utils:  xclip xdotool libnotify dunst xwallpaper bc
    Media: mpd mpc mpv ncmpcpp youtube-dl sxiv imagemagick
    Laptop: tlp xf86-input-synaptics

Configuration Notes: 
    XRANDR: xrandr --output <screen> --mode <resolution>
            xrandr --dpi 192
    XINIT:  /etc/X11/xinit/xinitrc ~/.xinitrc; replace last block with 
            eval $(/usr/bin/gnome-keyring/daemon --start --components=pkcs11,secrets,ssh)
            export SSH_AUTH_SOCK
            exec i3
    TERMITE:cp /etc/xdg/termite/config ~/.config/termite/
    HIDPI:  GDK_SCALE=2 in .profile
    
AUR Helper:
    YAY:    git clone 'https://aur.archlinux.org/yay.git'
            cd yay && makepkg -si
